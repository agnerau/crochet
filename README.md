# CROCHET FILET PATTERN GENERATOR

AI-driven generator for crochet filet patterns to make your crocheting life easier :3

- Uses OpenAI **GPT-5.1** model
- Max **15x15** squares
- Pattern can be downloaded as **PDF** or **PNG**
